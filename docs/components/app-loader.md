# MudExAppLoader

Application loading indicator with customizable appearance and behavior.

## Features

- ✅ **Customizable** - Control appearance and behavior
- ✅ **Multiple Styles** - Various loading indicator styles
- ✅ **Overlay Support** - Full-screen or inline display

## Basic Usage

```razor
<MudExAppLoader />
```

## With Custom Message

```razor
<MudExAppLoader LoadingMessage="Loading application..." />
```
