# Other Components

Additional utility components provided by MudBlazor.Extensions.

## MudExTextField

Enhanced text field with additional features.

```razor
<MudExTextField @bind-Value="@text" />
```

## MudExCultureSelect

Culture and language selection component.

```razor
<MudExCultureSelect @bind-Culture="@selectedCulture" />
```

## MudExAudioPlayer

Audio playback component.

```razor
<MudExAudioPlayer Source="@audioUrl" />
```

## MudExSplitPanel

Split panel layout for resizable areas.

```razor
<MudExSplitPanel>
    <LeftPanel>Content 1</LeftPanel>
    <RightPanel>Content 2</RightPanel>
</MudExSplitPanel>
```

## MudExHtmlEdit

HTML editing component.

```razor
<MudExHtmlEdit @bind-Value="@htmlContent" />
```
