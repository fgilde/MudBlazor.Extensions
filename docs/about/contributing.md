# Contributing

Thank you for your interest in contributing to MudBlazor.Extensions!

## How to Contribute

1. **Fork the Repository**
   - Fork the [MudBlazor.Extensions repository](https://github.com/fgilde/MudBlazor.Extensions)

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Your Changes**
   - Write clean, maintainable code
   - Follow existing code style
   - Add tests for new features

4. **Test Your Changes**
   - Ensure all tests pass
   - Test in multiple browsers if applicable

5. **Submit a Pull Request**
   - Describe your changes
   - Reference any related issues

## Reporting Issues

Found a bug? Have a feature request?

- Check [existing issues](https://github.com/fgilde/MudBlazor.Extensions/issues)
- Create a new issue with detailed information
- Include code samples and steps to reproduce

## Code of Conduct

Be respectful and inclusive in all interactions.

## Getting Help

- Read the [documentation](https://fgilde.github.io/MudBlazor.Extensions/)
- Check [discussions](https://github.com/fgilde/MudBlazor.Extensions/discussions)
- Ask questions in issues

## Thank You!

Your contributions make this project better for everyone. ❤️
